## How to use : 
**With npm :**
``npm i -g nginx-gui``

Then run : 
``nginx-gui`` and access http://localhost:9004

Or :
 
``nginx-gui --start-nginx`` for an auto start of nginx
## DEV MODE : 

- run ``npm start``
- run ``npm run server:dev``

- open [http://localhost:8080/webpack-dev-server/](http://localhost:8080/webpack-dev-server/)
